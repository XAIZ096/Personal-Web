# Short narrated video script

Use this as a 1-2 minute recording script for the public video requirement.

Hi, this is my Project 1 personal homepage. The site is built with vanilla HTML, CSS, and ES6 modules. It has three pages: the homepage, the about page, and a third AI generated page.

On the homepage, I introduce myself, show a few meaningful project categories, and include a project filter. The filter is original JavaScript and lets visitors show all projects or only web, hardware, or data projects. I also added a study focus generator as a creative component.

The about page includes my technical interests, a learning timeline, and a front-end only contact preview form. The form does not send data to a server; it only creates a preview on the page.

The AI generated page includes a study planner concept. A visitor can enter a topic and select a study time, then the page generates a small plan. I also included a GenAI disclosure explaining how AI was used.

For organization, CSS is in the assets/css folder, JavaScript modules are in the assets/js folder, and images are in the assets/images folder. The project also includes a README, design document, package.json, ESLint config, Prettier config, and MIT license.
